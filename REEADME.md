Instruções:

Para criar um ponto, utilize o botão DIREITO do mouse.

Para arrastar um ponto, utilize o botão ESQUERDO do mouse.

Para alterar entre diferentes curvas de bezier, aperte um número entre 0 e 9.

Para deletar um ponto, segure d e clique com botão ESQUERDO do mouse sobre ele.

Para gerar a transformação, após criadas algumas curvas, aperte b.
